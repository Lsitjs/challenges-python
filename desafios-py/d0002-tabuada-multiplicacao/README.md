# :heavy_check_mark: Desafio 0002: Tabuada de multiplicação

## Enunciado:

Usuária digita um número inteiro e programa retorna a tabuada de multiplicação desse.

---

**Entrada**

```
7
```

**Saida**

```
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
```
